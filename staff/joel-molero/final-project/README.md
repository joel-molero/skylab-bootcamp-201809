## How do I use the app?
--------
### To start the API
```cd ./atlantis-api; node .```

### To start the APP
```cd ./atlantis-app; yarn start```

--------

**_IMPORTANT !_**

(before starting the app, be sure to open in browser the API url, to do that type in the url bar ```https://<your-private-ip>:5060```)
